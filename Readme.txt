To build the code from scratch use the makefile
Run "make" to build the Jar
It requires Maven to be installed

To execute the Jar file use the below command
Usage: "java -jar target/staticanalyser-1.0-SNAPSHOT.jar <Path To SDK> <Language of SDK>"

Use this script to plot the graphs. It reads its input as CSV files from the output folders
python3.9 plotGraph.py
